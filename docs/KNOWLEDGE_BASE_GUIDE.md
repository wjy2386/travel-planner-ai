### 2. 知识库检索工具

项目提供了3个专用的知识库检索工具：

#### 2.1 search_destination_guide - 目的地攻略检索

**用途**: 获取目的地的全面攻略（景点、美食、交通、住宿）

**使用场景**:
- 用户首次提到某个目的地
- 开始规划行程前
- 用户询问"X地有什么好玩的"

**工具调用示例**:
```python
search_destination_guide("东京")
search_destination_guide("京都")
search_destination_guide("巴黎")
```

#### 2.2 search_seasonal_info - 季节性信息检索

**用途**: 获取目的地的季节性信息（最佳游览时间、天气、节日）

**使用场景**:
- 规划行程时需要考虑季节因素
- 用户询问"X地几月去最好"
- 需要了解特定季节的特色活动

**工具调用示例**:
```python
search_seasonal_info("东京", "春季")
search_seasonal_info("京都", "秋季")
search_seasonal_info("巴黎", "冬季")
```

#### 2.3 search_travel_knowledge - 通用知识检索

**用途**: 获取特定领域的专业知识（美食、景点、文化、技巧）

**使用场景**:
- 用户询问特定领域（美食、景点、文化等）
- 需要获取实用的旅行建议和技巧
- 需要补充专业信息

**工具调用示例**:
```python
search_travel_knowledge("东京美食推荐")
search_travel_knowledge("京都历史文化")
search_travel_knowledge("巴黎交通攻略")
```

## 使用指南

### 步骤1: 初始化知识库

在首次使用前，需要运行初始化脚本导入官方网站内容：

```bash
python src/tools/initialize_knowledge_from_urls.py
```

**注意**:
- 脚本会自动处理URL导入
- 每个URL之间有1秒间隔，避免请求过快
- 部分网站可能因反爬机制导入失败，这是正常的

### 步骤2: 验证知识库导入

运行测试脚本验证知识库是否正常工作：

```bash
python tests/test_knowledge_status.py
```

**期望输出**:
```
检查知识库状态...

尝试搜索所有数据集（不指定table_names）...
✓ 找到 X 条结果
```

### 步骤3: 测试Agent

使用 `test_run` 测试Agent的知识库检索能力：

```python
from coze_coding_dev_sdk import test_run

test_run(params="请帮我规划一个东京5天4晚的行程")
```

**期望行为**:
- Agent自动调用 `search_destination_guide` 获取东京攻略
- Agent调用 `search_seasonal_info` 了解季节信息
- Agent基于知识库生成专业的行程方案
- 输出中包含信息来源标注

## 知识库检索策略

### 检索参数配置

- **top_k**: 3-8（返回最相关的3-8条结果）
- **min_score**: 0.5-0.6（相关度阈值，过滤低质量结果）
- **数据集**: 搜索所有数据集（不指定table_names）

### 检索优化建议

1. **查询构建**: 使用具体、描述性的查询词
   - ✅ 好: "东京春季 最佳游览时间 天气"
   - ❌ 差: "东京"

2. **多关键词**: 使用空格分隔多个关键词
   - ✅ 好: "东京 美食 寿司 推荐"
   - ❌ 差: "东京美食"

3. **组合检索**: 针对不同场景使用不同工具
   - 攻略获取 → `search_destination_guide`
   - 季节信息 → `search_seasonal_info`
   - 专项知识 → `search_travel_knowledge`

## 输出规范

### 信息来源标注

在Agent输出中，必须标注信息来源：

```
浅草寺：东京最古老的寺庙，建议9点前到达避开人流（根据攻略指南）
```

### 相关度使用

优先使用相关度高的内容（评分>0.7），对低相关度内容进行二次验证。

### 知识库+搜索结合

当知识库信息不足时，结合联网搜索补充，并在输出中说明：

```
东京塔开放时间：9:00-23:00（根据攻略指南，实时信息已通过搜索验证）
```

## 常见问题

### Q1: 为什么有些URL导入失败？

A: 可能原因：
- 网站有反爬机制
- 网站暂时无法访问
- 网站结构发生变化

**解决方案**: 
- 检查URL是否正确
- 尝试手动访问URL
- 使用其他替代网站

### Q2: 知识库检索返回空结果？

A: 可能原因：
- 知识库中无相关内容
- 查询词不够具体
- 相关度阈值过高

**解决方案**:
- 降低 `min_score` 参数
- 使用更具体的查询词
- 检查知识库是否成功初始化

### Q3: Agent没有使用知识库？

A: 检查：
1. 知识库工具是否正确集成到 `itinerary_agent.py`
2. System Prompt是否明确知识库使用策略
3. 工具参数是否正确
