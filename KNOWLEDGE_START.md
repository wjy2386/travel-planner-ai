## 🛠️ 常见操作

### 添加新目的地

```python
from tools.init_knowledge import add_custom_knowledge

content = """
大理旅游攻略

【最佳游览时间】
春季（3-4月）：春暖花开，苍山洱海风光迷人

【必游景点】
大理古城、洱海、苍山、双廊古镇

【美食推荐】
大理砂锅鱼、酸辣鱼、乳扇、白族三道茶
"""

add_custom_knowledge(content, "大理")
```

### 使用CLI添加知识

```bash
coze-coding-ai knowledge add \
  --dataset "travel_knowledge" \
  --content "你的知识内容"
```

### 搜索知识库

```bash
coze-coding-ai knowledge search \
  --query "东京美食" \
  --top-k 3
```
